# Cric-View
A simple React app to view live cricket matches with search, filter and sorting features.

##  About Project
This is a web app where we can see live cricket matches and match details.  
User can search matches, filter them and also sort the data.

## Purpose
Main goal of this project is to learn JavaScript, React and API calling and how to show data on website.

## 🔗 API Used
CricketData API  
https://cricketdata.org/

### Core Features
- Show live cricket matches  
- Search matches using team name  
- Filter matches (ODI, T20, Test)  
- Sort matches by date or score  

### Extra Features
- Dark mode option  
- Add matches to favorite  

##  Tech Used
- HTML  
- CSS  
- JavaScript  
- React  

##  How to Run
1. Clone the project  
2. Run npm install  
3. Run npm start
